Bind it!
1.	Create three new components: GameControl, Odd and Even
2.	The GameControl Component should have buttons to start and stop the game
3.	When starting the game, an event (holding a incrementing number) should get emitted each second (ref = setlntervalO)
4.	The event should be listenable from outside the component

Свяжи это!
1. Создайте три новых компонента: GameControl, Odd и Even.
2. Компонент GameControl должен иметь кнопки для запуска и остановки игры.
3. При запуске игры событие (содержащее возрастающее число) должно генерироваться каждую секунду (ref = setlntervalO).
4. Событие должно быть доступно для прослушивания снаружи компонента.
5. При остановке игры больше не должно генерироваться событий (clearInterval(ref))
6. Новый нечетный компонент должен создаваться для каждого выпущенного нечетного числа, то же самое должно произойти с четным компонентом (для четных чисел).
7. Просто выведите нечетное число или четное число в двух компонентах.
8. Оформите элемент (например, абзац), содержащий выводимый текст, по-разному в обоих компонентах.
